Star Trek II Quiz (with JavaScript)

Treehouse

This project was done along with a tutorial from Treehouse. They provided some of the styling. This was more of a beginner level project with JavaScript.

The Project

I learned some foundational JavaScript element manipulation, while applying my deep and thorough love and knowledge of Star Trek II: The Wrath of Khan (probably the greatest film ever made). Using a simple array to store the questions, it is very easy to add, delete or edit them.
